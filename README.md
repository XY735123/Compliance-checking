# Compliance-checking
## INSTALLATION
### create a virtual environment to install the packages in and run the project:
python3 -m venv .project_env
### install necessary packages in the right versions within virtual env.:
pip install -r requirements.txt
### install spacy en_core_web_lg (https://spacy.io/usage)
pip install https://github.com/explosion/spacy-models/releases/download/en_core_web_lg-2.0.0/en_core_web_lg-2.0.0.tar.gz
